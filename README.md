# 潮飨·知味 (Chaoxiang Zhiwei)

## 项目简介
潮飨·知味是首个融合饮食文化认知计算与个性化味觉图谱的美食智能体，让AI真正懂得"潮汕味道"。

## 核心功能
- **文化语境理解**：构建"时令-地域-技法-典故"四维认知框架。
- **动态味觉图谱**：持续构建并优化个人味觉画像。
- **时空耦合推荐**：整合时令、地理和社交情绪的精准推荐。

## 目录结构
- `agent/`: Agent核心模块
- `tools/`: 工具服务
- `data/`: 美食数据库
- `ui/`: Gradio界面
- `config/`: 配置
- `tests/`: 测试代码
- `docs/`: 文档
